Drag&Drop auf der Hauptkarte
   
DS Workbench bietet seit Version 2.0 eine Vielzahl von Möglichkeiten, um Dörfer von der Hauptkarte per Drag&Drop an viele Stellen zu übertragen. Die Handhabung ist denkbar einfach. Man fährt mit der Maus ohne gewähltes Werkzeug über ein Dorf und zieht das Dorf mit gedrückter linker Maustaste an die Stelle, wo man es einfügen möchte. Sind mehrere Dörfer auf der Karte markiert werden alle markierten Dörfer erfasst und können auf einmal bewegt werden. Der Mauscursor verwandelt sich dabei in ein Dorfsymbol, welches anzeigt, wieviele Dörfer gerade mit der Maus bewegt werden. Die folgenden Komponenten/Ansichten in DS Workbench erlauben es, dass man auf sie Dörfer zieht:
 
Hauptkarte Zieht man auf der Hauptkarte eins oder mehrere ausgewählte Dörfer auf ein anderes Dorf, so können Angriffe, ausgehend von den erfassten Dörfern, auf das Zieldorf geplant werden. Es erscheint der bekannte Angriff hinzufügen Dialog, bei mehreren gewählten Dörfern ist unter Herkunft die Anzahl der angreifenden Dörfer vermerkt. Die eingetragene Laufzeit basiert dabei auf dem am weitesten entfernten Dorf.
Angriffsplaner Dörfer können direkt von der Karte in die Herkunfts- oder Zieltabelle des Angriffsplaners gezogen werden.
Auswahlübersicht Die bisherige Funktion der Auswahlübersicht wurde insofern abgeändert, dass Dörfer die auf der Karte mit dem Auswahlwerkzeug markiert werden nicht mehr direkt in die Auswahlübersicht eingetragen werden, sondern dorthin auf Wunsch per Drag&Drop verschoben werden können.

Entfernungsübersicht

 Wie schon im Angriffsplaner können Dörfer direkt in die Tabelle der Entfernungsübersicht gezogen werden, woraufhin sie darin als neue Spalten auftauchen.
Manueller Angriffsplaner Beim manuellen Angriffsplaner können einzelne Dörfer in das Herkunft- oder Zielfeld gezogen werden.
Notizblock Markierte Dörfer können im Notizblock in die Liste der zugehörigen Dörfer gezogen werden. Dabei ist es notwendig, dass man bereits eine neue Notiz erstellt bzw. eine bestehende Notiz ausgewählt hat.
Tags Übersicht Zieht man markierte Dörfer in die Tags Übersicht, so werden den Dörfern alle Tags zugewiesen, die man in der Tags-Tabelle ausgewählt hat.
