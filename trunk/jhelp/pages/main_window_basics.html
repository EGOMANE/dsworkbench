Funktionen im Hauptfenster

   
In Hauptfenster befindet sich auf der rechten Seite unter der Minimap verschiedene Aufgabengruppen, welche die folgenden Funktionen beinhalten:
 

Navigation
 

Hier findet man die Elemente, die der Navigation auf der Hauptkarte dienen. Es kann eine bestimmte Position zentriert, Bildschirmweise gescrollt oder gezoomt werden. Optional kann man das aktuell zentrierte Dorf im Spiel zentriert anzeigen lassen.

 

Allgemeine Informationen

 
In diesem Bereich findet man zum einen Informationen über den aktuell gewählten Spieler und Server und das momentan aktive Spielerdorf, zum anderern findet man Informationen über den Onlinestatus und das momentan gewählte Werkzeug.
Weiterhin sind in dieser Aufgabengruppe Funktionen zum Zentrieren des momentan gewählten Spielerdorfes im Spiel enthalten oder zum Erstellen eines Fotos der Hauptkarte. Für diese Fotos kann man weiterhin entscheiden, ob sie auf der lokalen Festplatte gespeichert oder Online gestellt werden sollen. Stellt man sie Online, so wird die aktuelle Kartenansicht auf den DS Workbench Server hochgeladen und ist über eine eindeutige ID im Browser aufrufbar.
 
Für Online gestellt Fotos existiert eine Größenbegrenzung auf 200 KB. Bekommst du eine Fehlermeldung, dass das hochzuladende Bild zu groß ist, versuche entweder das DS Workbench Fenster zu verkleinern, einen anderes Grafikpaket zu wählen ('Minimap' sollte in jedem Fall funktionieren) oder den Zoomfaktor zu verändern.
 

Kartenanzeige
 
Die darin enthaltenen Funktionen dienen der Modifikation der Kartendarstellung. Hier kann man das verwendete Grafikpaket direkt auswählen oder das Kartenpopup, welches erscheint wenn man mit der Maus über ein Dorf auf der Karte fährt, (de-)aktivieren. Außerdem kann man angeben, welche Art von Truppen eines Dorfes im Popup dargestellt werden. Weiterhin enthält diese Aufgabengruppe die Einstellung des Laufzeitradars, das über das entsprechende Werkzeug auf ein Dorf gesetzt werden kann, um Dörfer in der Reichweite des angegebenen Radius zu finden.
Zuletzt bietet dieser Bereich die Möglichkeit, die Reihenfolge der gezeichneten Ebenen zu bestimmen:
 
 
Hier kann man über die Buttons auf der rechten Seite die verschiedenen zu zeichnenden Ebenen in den Hintergrund (weiter oben) oder den Vordergrund (weiter unten) verschieben. Eine besondere Rolle spielen dabei die Ebene mit dem Namen 'Dörfer', die rot unterlegt gekennzeichnet ist und die Ebene 'Markierungen ', die grau unterlegt ist. Diese beiden Ebenen werden immer gezeichnet. Verschiebt man die Ebene 'Markierungen' unter die Ebene 'Dörfer', so erreicht man, dass Markierungen über den Dörfern gezeichnet werden, was dem früheren MarkOnTop Modus entspricht, der dazu dient, eine schnelle Übersicht über Stammes- und Spielerverteilungen zu erhalten.
Alle anderen Ebenen werden nur gezeichnet, wenn sie unterhalb der 'Dörfer' Ebene liegen. Im oberen Berispiel würde die Ebene 'Truppendichte' nicht gezeichnet werden. Das Verschieben der Ebenen kann man zum einen verwenden, um einzelne Ebenen deutlicher darzustellen indem man sie in den Vordergrund verschiebt. Zum anderen kann man eine oder mehrere Ebenen, durch ihre Positionierung oberhalb der 'Dörfer' Ebene, 'unsichtbar' machen und so die Zeichengeschwindigkeit in DS Workbench erhöhen, besonders wie oben gezeigt für das Zeichnen der Truppendichte.
 

ROIs
 
An dieser Stellen können sogenannte Regions of Interest (ROIs) erstellt und verwaltet werden. Eine ROI ist genaugenommen eine Koordinate auf der Karte, die von besonderem Interesse ist. (mehr...)
 

UV Einstellungen
 
Die UV-Einstellungen dienen dazu, InGame Aktionen mit dem UV-Account durchführen zu können. Im Gegensatz zum Spieleraccount unterscheiden sich die URLs eines UV-Accounts wenn sie im Browser aufgerufen werden um einen zusätzlichen Wert t, wie er im folgenden gezeigt ist:
 
http://de0.die-staemme.de/game.php?t=743256&village=269739&screen=place
 
Dies ist die UV-ID die man in das entsprechende Feld im DS Workbench Hauptfenster eintragen muss, um auf den Versammlungs- und den Marktplatz des UV-Accounts zuzugreifen. Den DS Workbench UV-Modus aktiviert man über den neben dem Eingabefeld befindlichen Button. Ist der Button nicht aktiviert, wie im Screenshot oben zu sehen, werden InGame Aktionen auf den eigenen Account angewendet.
