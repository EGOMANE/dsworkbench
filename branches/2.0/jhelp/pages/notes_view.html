Notizblock
 
 

Mit dem Notizblock können Notizen zu einzelnen oder mehreren Dörfern erstellt werden. Für welche Dörfer eine Notiz gilt sieht man in der Liste auf der linken Seite. Erstellt man eine Notiz über das entsprechende Werkzeug, so wird das gewählte Dorf automatisch in diese Liste übernommen. Ansonsten besteht die Möglichkeit, einzelne Dörfer nachträglich hinzuzufügen. Das kann man auf zwei verschieden Arte bewerkstelligen:

  1. Man klickt auf der Hauptkarte ohne gewähltes Werkzeug auf ein Dorf. Dessen Koordinaten werden automatisch in das Eingabefeld unter der Dorfliste übernommen und können über einen Klick auf den Button mit dem weißen Plus in die Liste eingefügt werden.
  2. Man gibt Dorfkoordinaten einzeln in das Eingabefeld ein (manuell oder per Copy&Paste) und fügt das Dorf über den entsprechenden Button hinzu. Dabei muss der Wert auf Karten mit dem XY-Koodinatensystem dem folgdenden Format entsprechen: Dorfname (X|Y), auf Karten mit hierarchischem Koodinatensystem diesem Format: Con:Sec:Sub

Über die anderen beiden Buttons neben der Dorfliste kann man einzelne Einträge löschen oder auf der Hauptkarte zentrieren. Die eigentlichen Notizfunktionen befinden sich im rechten Teil des Notizblocks. Über dem Eingabefeld für den Notiztext hat man die Möglichkeit zu erkennen, wann die entsprechende Notiz das letzte Mal geändert wurde. Zudem kann man das Symbol einstellen, mit dem die Notiz auf der Karte gekennzeichnet wird, sowie das Symbol, mit dem die Notiz selbst im Kartenpopup versehen wird, wie nachfolgend zu sehen.

 
 
Unter dem Eingabefeld für den Notiztext hat man auf der linken Seite die Möglichkeit, durch die Notizen zu navigieren, die aktuell gewählte Notiz zu löschen oder eine neue Notiz zu erstellen. Auf der rechten Seite findet man die Suchfunktion. Hier kann man in das Eingabefeld den Text eintragen, nach dem in allen Notizen gesucht werden soll. Ein Klick auf den Suchen-Button startet die Suche und springt zur nächsten Notiz, bei der der Notiztext ODER der Name eines Dorfes den zu suchenden Begriff enthält. Gesucht wird dabei ausgehend von der gewählten Notiz. Klickt man anschließend also nochmal auf den Suchen-Button, so wird die nächste Notiz gesucht, die dem Suchkriterium entspricht.
