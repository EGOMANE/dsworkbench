Gefilterte Auswahl von Angriffen
 
 
Über die gefilterte Auswahl von Angriffen kann genau bestimmt werden, welche Angriffe ausgewählt werden sollen. Der Filterdialog ist in zwei Bereiche aufgeteilt. Im oberen Bereich kann man aus einer List einen angreifenden Spieler auswählen, woraufhin dessen Dörfer angezeigt werden, aus denen Angriffe im aktuell gewählten Angriffsplan existieren. Nun kann man über den Button Alle die Dörfer des Spielers auswählen, um nur Angriffe die von diesem Spieler kommen, zu markieren. Natürlich ist es auch möglich, nur einzelne Dörfer auszuwählen.
Im unteren Bereich gibt es dieselben Möglichkeiten für Spieler und Dörfer, auf die Angriffe geplant sind. Weiterhin kann man Herkunft und Ziel kombinieren, um Angriffe zwischen einzelnen Spielern und Dörfern zu filtern.
 
Es ist zur Zeit, sowohl für die Herkunft als auch für das Ziel, nicht möglich mehrere Spieler gleichzeitig anzugeben. Dies wird eventuell in einer kommenden Version hinzugefügt.
